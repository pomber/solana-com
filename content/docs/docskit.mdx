## Code Marks

Highlighting lines with default and custom colors:

```js
function lorem(ipsum, dolor = 1) {
  // !mark
  return dolor;
}

function ipsum(lorem, dolor = 1) {
  // !mark(1:2) gold
  const sit = lorem == null ? 0 : 1;
  dolor = sit - amet(dolor);
  return sit ? consectetur(lorem) : [];
}
```

Highlighting tokens using regex:

```js
// !mark[/sit/mg] #d946ef
function ipsum(lorem, dolor = 1) {
  // !mark[/lorem == null/]
  const sit = lorem == null ? 0 : 1;
  dolor = sit - amet(dolor);
  return sit ? consectetur(lorem) : [];
}
```

Highlighting diffs:

```js
function lorem(ipsum, dolor = 1) {
  const sit = 10;
  // !diff -
  dolor = ipsum - sit;
  // !diff +
  dolor = sit - amet(dolor);
  return sit ? consectetur(ipsum) : [];
}
```

## Collapse

Collapsing groups of lines:

```js title="index.js"
// !collapse(1:4)
function lorem(ipsum, dolor = 1) {
  const sit = ipsum == null ? 0 : 1;
  dolor = sit - amet(dolor);
  return sit ? consectetur(ipsum) : [];
}

// !collapse(1:4) collapsed
function ipsum(ipsum, dolor = 1) {
  const sit = ipsum == null ? 0 : 1;
  dolor = sit - amet(dolor);
  return sit ? consectetur(ipsum) : [];
}
```

Folding tokens using regex:

```jsx
// !fold[/className="(.*?)"/gm]
function Foo() {
  return (
    <div className="bg-red-200 opacity-50">
      <span className="block">hey</span>
    </div>
  );
}
```

## Inline Code

This is a variable in JavaScript: _`var x = 10`_.

In CSS it looks like this: _css`a { --x: 10 }`_.

And in Pascal it's _pascal`x := 10`_.

## Mentions

<WithMentions>

The [base case](mention:one) returns 1.

```cpp
int factorial(int n) {
  if (n == 0) {
    // !mention one
    return 1;
  } else {
    // !mention two
    return n * factorial(n - 1);
  }
}
```

The [recursive case](mention:two) multiplies something.

</WithMentions>
